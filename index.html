Biography

    I'm a deep learning performance software engineer in the TensorRT team at NVIDIA. I received my Ph.D. in Computer Engineering from Purdue University in December 2021. I interned in the Systems Research Group at Microsoft Research Redmond in Summer 2017.

Research

    Computer systems: operating systems, compiler, architecture, and runtime systems for emerging applications and hardware.

  • At NVIDIA: working on TensorRT for high-performance deep learning inference on NVIDIA GPUs.

  • At Purdue: worked on system for machine learning and real-time data analytics on modern hardware, e.g., many-core CPUs, hybrid memory (DRAM + 3D-stacked memory), and tiny microcontrollers, aiming to improve system performance and enable new use cases.

  • At ICT-CAS: worked on OS kernel (memory subsystem & drivers), system virtualization (QEMU & KVM), and RPC systems (with RDMA).
