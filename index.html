Biography

    I'm a senior deep learning performance software engineer in the TensorRT team at NVIDIA. I received my Ph.D. in Computer Engineering from Purdue University in December 2021. I interned in the Systems Research Group at Microsoft Research Redmond in Summer 2017. I work on computer systems, with a recent focus on systems for machine learning.

Research

    Computer systems: operating systems, compiler, architecture, and runtime systems for emerging applications and hardware.

  • NVIDIA (2022 - now): TensorRT for high-performance deep learning inference on NVIDIA GPUs.

  • Purdue (2015 - 2021): system for machine learning and real-time data analytics on modern hardware, e.g., many-core CPUs, hybrid memory (DRAM + 3D-stacked memory), and tiny microcontrollers, aiming to improve system performance and enable new use cases.

  • ICT-CAS (2012 - 2015): OS kernel (memory subsystem & drivers), system virtualization (QEMU & KVM), and RPC systems (with RDMA).
